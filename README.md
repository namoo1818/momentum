# momentum
노마드 코더 강의 - 바닐라 JS로 크롬 앱 만들기 실습

# 프로젝트 목표
- 클론 코딩으로 JS 기초 이론 마스터
- HTML, CSS 복습
- 외부 API 활용

- 반응형 웹
- 랜덤 배경화면
- 실시간 시계
- localStorage를 활용한 로그인과 리셋
